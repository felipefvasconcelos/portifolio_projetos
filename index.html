Projetos em Ciência de Dados

Rankeamento Cross Sell de Seguros.

O objetivo deste projeto é ajudar uma empresa fictícia de seguros que deseja ofertar um novo produto em seu portfólio. O problema proposto é construir um modelo de rankeamento de clientes de acordo com sua propensão de compra de um novo produto que será ofertado pela empresa de seguros utilizando dados públicos do Kaggle. O modelo colocado em produção, com um precision at k de 0.35 e recall at k de 0.57 demonstou um perormance quase 3x melhor que um modelo aleatório quando trabalhando com 20% da base de clientes.

As ferramentas utilizadas foram:

  • Git e Github.
  • Python, Pandas, Sklearn, Scipy.
  • Jupyter Notebook.
  • Extra Trees Classifier como seletor de features.
  • Logistic Regression, KNeighbors Classifier (KNN), Gradiant Boosting Classifier, Random Forest Classifier, XGBoost Classifier.
  • Heroku

Previsão de Vendas Drogarias Rossmann.

O objetivo deste projeto é simular um problema de negócio da vida real e buscar a solução utilizando modelos de Machine Learning. O problema proposto é construir um modelo de previsão de vendas com visibilidade de seis semanas de um grupo de lojas da rede de drograrias Rossmann utilizando dados públicos do Kaggle. O modelo colocado em produção, com um MAPE (Mean Absolut Percentage Error) de 0.095, retornou um valor total de R$ 285.047.136 em vendas nas próximas seis semanas.

As ferramentas utilizadas foram:

  • Git e Github.
  • Python, Pandas, Sklearn, Scipy.
  • Jupyter Notebook.
  • Boruta como seletor de features.
  • Linear Regression, Lasso, Randon Forest Regressor e XGBoost Regressor.

Insights do Mercado Imobiliário de King County (USA)

Neste projeto eu utilizo conceitos básicos de Python para fornecer insights sobre o negócio imobiliário em King County, baseado em dados públicos disponíveis no Kaggle. O objetivo foi analisar os dados dos imóveis disponíveis para venda, classifica-los como "comprar" ou "não comprar" e, finalmente, sugerir o melhor preço para compra e para venda. A conclusão foi a recomendação de compra de 10.500 imóveis (num total de 21.421 disponíveis para venda), onde o investimento total na compra seria de $ 4.078.617.894 com lucro esperado mínimo e máximo de $ 694.433.672 e $ 839.540.926, respectivamente

As ferramentas utilizadas foram:

  • Git e Github.
  • Python, Pandas, Matplotlib e Seaborn.
